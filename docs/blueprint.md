# **App Name**: HopOn

## Core Features:

- User Authentication: Secure signup/signin flow with email/password and OAuth (Google) including email verification.
- Trip Management: Drivers can create, edit, and delete trip postings with details like origin, destination, date/time, seats, price, and vehicle info.
- Trip Search and Booking: Passengers can search for trips by origin/destination/date/time, view results on a map and list, apply filters, book seats, and cancel bookings.
- Real-Time Chat: In-app chat functionality for communication between drivers and booked passengers with push notifications.
- Driver Rating and Dispute Resolution: Reviews, ratings, and a dispute reporting system enable user feedback and moderation.
- Personalized Trip Suggestions: AI-powered tool which suggests personalized trips for passengers based on their saved searches, trip history and wallet status.
- Map Integration & Routing: Integration with Mapbox or Google Maps to display routes, ETA, and distance.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for trust and reliability, evoking a sense of security during travel.
- Background color: Very light blue (#F0F9FF), a desaturated shade of the primary color for a clean and calm backdrop.
- Accent color: Warm orange (#E67700), approximately 30 degrees to the left on the color wheel, used to highlight CTAs.
- Body font: 'PT Sans', a humanist sans-serif for clear, accessible readability.
- Headline font: 'Playfair', a modern serif with high contrast for an elegant and fashionable feel. Note: currently only Google Fonts are supported.
- Simple, intuitive icons to represent trip details, user profile elements, and actions. Ensure they meet accessibility contrast standards.
- A clean, minimal design that adapts to different screen sizes (mobile, tablet, desktop). Uses a grid layout for trip listings and a combined list + map view.
- Subtle transitions and animations for loading states, form validation, and user interactions (e.g., booking requests) to provide a smooth user experience.
